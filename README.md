Task 3 - Data Analysis Using Pandas

Objective :
Develop a Python application to analyze a CSV dataset using the Pandas library. The project demonstrates data loading, cleaning, filtering, statistical analysis, and data visualization.

Features :
- Load a CSV dataset using Pandas
- Display dataset information
- Calculate average marks of students
- Identify the top-performing student
- Generate summary statistics
- Visualize data using a bar chart
- Easy-to-understand output

Technologies Used :
- Python 3
- Pandas
- Matplotlib

Requirements :
  Install the required libraries :
  pip install pandas matplotlib

How to Run:
1. Open the project in Visual Studio Code.
2. Open the terminal.
3. Run the program using:
python data_analysis.py

Dataset Used :
The project uses a CSV file named "student_data.csv"
